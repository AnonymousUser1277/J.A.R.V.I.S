# J.A.R.V.I.S. (Just A Rather Very Intelligent System)

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?style=for-the-badge&logo=python&logoColor=white) ![Platform](https://img.shields.io/badge/Platform-Windows-0078D6?style=for-the-badge&logo=windows&logoColor=white) ![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge) ![Status](https://img.shields.io/badge/Status-Active-success?style=for-the-badge)

> **A fully autonomous, context-aware, voice-activated AI assistant for Windows.**

JARVIS is not just a chatbot; it is a complete system controller. It integrates State-of-the-Art LLMs (Groq, Cohere, Gemini) with local automation to manage your PC, analyze your screen, secure your system with Face ID, and even sync with a mobile companion app.

## ğŸ› ï¸ Installation

### Prerequisites
1.  **Python 3.10+** installed.
2.  **Google Chrome** installed.
3.  **Tesseract OCR** installed (Add to Path or configure in Settings).
4.  **Redis Server** (for caching) installed and running on localhost.

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/JARVIS.git
cd JARVIS
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```
*Note: You may need to install `winshell` and `pyaudio` dependencies separately depending on your environment.* 

### 3. First Run & Setup
Run the included batch file to start JARVIS with Administrator privileges (required for system automation):

```cmd
Jarvis.bat
```

**On the first launch:**
1.  **Setup Wizard:** A GUI will appear guiding you through API Key configuration (Groq, Cohere, Gemini, etc.) and system preferences.
2.  **Face Registration:** JARVIS will ask to register your face for future authentication.

## ğŸš€ Usage

### Controls
*   **Voice:** Say *"Jarvis"* to wake him up.
*   **Orb UI:**
    *   **Left Click:** Toggle listening manually.
    *   **Right/Drag:** Move the orb.
*   **Hotkeys:**
    *   `Win + Space`: Open text input dialog.
    *   `Win + Enter`: Toggle microphone.
    *   `Alt + Shift + C`: View code generated by the last command.

### Common Commands
*   *"Open Chrome and search for latest tech news."*
*   *"Look at my screen and explain this code."*
*   *"Generate a realistic image of a cyberpunk city."*
*   *"Create a report about my system battery life."*
*   *"Send an email to boss@example.com saying I will be late."*
*   *"Switch to the next slide."* (Simulates keypresses)
